Click the refresh button to create new random circles.

In this assignment we learned to use HTML5's canvas and manipulate it using JS.

In this example we set the the canvas element to a var in JS and then set it's height and width. Then we set the canvas's 2D 'context' to the var 'c'. You can think of the context like a paintbrush. It then makes a for loop that creates 100 circles using the Math.random() to randomize the radii and positions of circle. We also used Math.Random() to randomize a hex code to represent a random color for each circle.
